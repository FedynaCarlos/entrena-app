# AI CONTEXT

## Proyecto

Nombre: Entrena App

Aplicación desarrollada con React para gestionar entrenamientos personalizados.

El objetivo es construir un MVP funcional y escalable.

No es un proyecto académico.

Es un producto real que será utilizado por alumnos.

---

## Objetivo del MVP

Permitir que un entrenador:

- cree ejercicios
- cree rutinas
- asigne rutinas

Y que un alumno:

- vea el entrenamiento del día
- complete los ejercicios

---

## Filosofía

Siempre priorizar:

- simplicidad
- escalabilidad
- mantenibilidad
- experiencia del usuario

Nunca agregar complejidad innecesaria.

---

## Stack

Frontend

- React
- Vite
- React Router

Backend (futuro)

- Node.js
- Express
- MongoDB

---

## Arquitectura

La aplicación está organizada por responsabilidades.

No crear componentes que todavía no sean utilizados.

Cada componente debe tener una única responsabilidad.

Trabajamos por Sprint.

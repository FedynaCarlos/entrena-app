# Entrena-App 🚀

Plataforma web para entrenamiento físico personalizado.

Entrena-App nace con el objetivo de acompañar a personas que quieren mejorar su salud y condición física mediante rutinas guiadas, seguimiento y una experiencia cercana con el entrenador.

## 🎯 Objetivo del proyecto

Crear una plataforma simple y escalable donde los usuarios puedan:

* Acceder a rutinas de entrenamiento.
* Seguir planes según su nivel.
* Sentirse acompañados durante el proceso.
* Registrar su progreso.
* Recibir una experiencia personalizada.

## 🏋️ Enfoque del MVP

La primera versión estará enfocada en validar el modelo:

* Catálogo de rutinas.
* Diferentes niveles:

  * Inicial
  * Intermedio
  * Avanzado
* Visualización de entrenamientos.
* Perfil de usuario.
* Sistema preparado para futuras integraciones.
